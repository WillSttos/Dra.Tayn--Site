# **App Name**: Blank Canvas

## Core Features:

- Project Creation: Allow users to create new, empty projects to begin their work.
- Basic UI Elements: Provide a palette of basic UI components that the user can place into the blank project.
- Element Positioning: Enable users to drag and drop components, position, and resize them within the project area.
- Layout Management: Basic layout management feature using a grid layout system or similar to assist in structuring the UI.
- Download Prototype: Generate a basic set of HTML/CSS to download.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calm, creative atmosphere.
- Background color: Very light gray (#F5F5F5) to create a subtle contrast with UI elements.
- Accent color: Light blue (#ADD8E6) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern look, suitable for headlines and body text
- Use simple, clear icons to represent UI elements and actions.
- Clean, minimalist layout that allows the user's prototype to be the central focus.
- Subtle transitions for dragging and dropping elements to provide a smooth user experience.
